💰 Personal Expense Tracker

A full-stack web application to help users track, manage, and analyze daily expenses efficiently. Built using PHP, MySQL, and Bootstrap, this project provides a simple and user-friendly interface for managing personal finances.

🚀 Features  <br>
👤 User Registration & Login<br>
🔐 Secure Authentication System<br>
➕ Add Daily Expenses<br>
📊 View Expense Reports<br>
🗂️ Categorize Expenses<br>
✏️ Update & Delete Expenses<br>
👤 Profile Management<br>
🔒 Change Password<br><hr>
🛠️ Tech Stack<br>
Frontend: HTML, CSS, Bootstrap<br>
Backend: PHP<br>
Database: MySQL<br>
Server: XAMPP (Apache + MySQL)<br>
